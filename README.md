# 📌 Gerenciador de Tarefas (React Intermediário)

## 📖 Descrição
O **Gerenciador de Tarefas** é uma aplicação web desenvolvida em **[React](ca://s?q=React_JS)** que permite ao usuário **adicionar, listar e excluir tarefas**.  
O projeto utiliza **[Context API](ca://s?q=Context_API_em_React)** para gerenciamento de estado global e **[React Router](ca://s?q=React_Router)** para navegação entre páginas, demonstrando boas práticas de organização e componentização em nível intermediário.

---

## 🚀 Funcionalidades
- **[Adicionar tarefas](ca://s?q=Adicionar_tarefas_em_React)** com formulário simples  
- **[Listar tarefas](ca://s?q=Listar_tarefas_em_React)** em tempo real  
- **[Excluir tarefas](ca://s?q=Excluir_tarefas_em_React)** com botão dedicado  
- **[Navegação multipáginas](ca://s?q=Navegação_com_React_Router)** (Home, Sobre, Configurações)  
- **[Gerenciamento de estado global](ca://s?q=Gerenciamento_de_estado_em_React)** via Context API  

---

## 🛠️ Tecnologias utilizadas
- [React](ca://s?q=React_JS)  
- [React Router](ca://s?q=React_Router)  
- [Context API](ca://s?q=Context_API_em_React)  
- [CSS](ca://s?q=Estilização_em_CSS) para estilização básica
