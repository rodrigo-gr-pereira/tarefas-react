# 📌 Gerenciador de Tarefas (React + Tailwind v4 + Dark Mode)

## 📖 Descrição
O **Gerenciador de Tarefas** é uma aplicação web desenvolvida em **[React](ca://s?q=React_JS)** com **[TailwindCSS v4](ca://s?q=TailwindCSS_v4)**.  
Ele permite ao usuário **adicionar, listar e excluir tarefas**, além de alternar entre **tema claro e escuro** com suporte nativo ao **[Dark Mode](ca://s?q=Dark_mode_em_TailwindCSS)**.  
O projeto utiliza **[Context API](ca://s?q=Context_API_em_React)** para gerenciamento de estado global e **[React Router](ca://s?q=React_Router)** para navegação multipáginas.

---

## 🚀 Funcionalidades
- **[Adicionar tarefas](ca://s?q=Adicionar_tarefas_em_React)** com formulário estilizado  
- **[Listar tarefas](ca://s?q=Listar_tarefas_em_React)** em tempo real  
- **[Excluir tarefas](ca://s?q=Excluir_tarefas_em_React)** com botão dedicado  
- **[Navegação multipáginas](ca://s?q=Navegação_com_React_Router)** (Home, Sobre, Configurações)  
- **[Dark Mode](ca://s?q=Dark_mode_em_TailwindCSS)** com alternância manual (☀️ / 🌙)  
- **[Gerenciamento de estado global](ca://s?q=Gerenciamento_de_estado_em_React)** via Context API
